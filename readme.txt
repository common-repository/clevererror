CleverError intelligently replace your old 404 'Page Not Found'-page with a supercharged version with an internal site search.

== Description ==

Rather than losing a visitor when they mistype a URL, follow an old link or end up where they shouldn't you provide them with an easy and elegant way to find their way back to your site and content.

If you wish to learn a bit more about CleverError - check out http://clevererror.com

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `clevererror`-folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You can do further customizations in the `CleverError`-menu
